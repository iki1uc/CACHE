# CACHE · MISS/FAIL/RDY · NC-Fehlerstation

CACHE ist die Fehler- und Erfolgsstation des iki1uc-Systems.
Sie bewertet Eingänge, erzeugt MISS/FAIL/RDY und konvertiert Werte in NC-Modi.

──────────────────────────────────────────────
## Achsen

- MISS – verfehlt
- FAIL – kaputt
- RDY – bereit
- SCORE – Bewertung

──────────────────────────────────────────────
## Modi

- 6e → 6
- 12e → 6e
- RAW → 6d

──────────────────────────────────────────────
